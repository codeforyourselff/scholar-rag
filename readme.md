# Scholar RAG

Scholar RAG is a simple web application for research and academic assistance using retrieval-augmented generation (RAG) techniques.

## Features

- Search and retrieve relevant academic content
- Generate summaries or answers using contextual retrieval
- Simple and lightweight web interface

## Getting Started

1. Clone the repository.
2. Install dependencies.
3. Start the development server.

## Project Structure

- `readme.md` - Project documentation
- `src/` - Application source code
- `public/` - Static assets
